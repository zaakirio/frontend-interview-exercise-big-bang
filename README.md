## Getting Started

### Installation

Make sure you have pnpm installed on your system. If not, you can install it with:

```bash
npm install -g pnpm
```

1. Install dependencies:
```bash
pnpm install
```

2. Run Application:
```bash
pnpm dev
```

3. Start the development server:
```bash
pnpm dev
```
